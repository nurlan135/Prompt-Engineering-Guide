# Giriş

Prompt mühəndislik müxtəlif tətbiqlər və tədqiqat mövzuları üçün dil modellərindən (LM) səmərəli istifadə etmək üçün göstərişlərin işlənib hazırlanması və optimallaşdırılması üçün nisbətən yeni bir fəndir. Prompt mühəndislik bacarıqları böyük dil modellərinin (LLM) imkanlarını və məhdudiyyətlərini daha yaxşı başa düşməyə kömək edir. Tədqiqatçılar suallara cavab vermək və arifmetik əsaslandırma kimi geniş spektrli ümumi və mürəkkəb tapşırıqlar üzrə LLM-lərin imkanlarını artırmaq üçün prompt mühəndislikdən istifadə edirlər. Tərtibatçılar LLM-lər və digər alətlərlə əlaqə quran möhkəm və effektiv sövq üsullarını tərtib etmək üçün prompt mühəndislikdən istifadə edirlər.

Bu təlimat LLM-lərlə qarşılıqlı əlaqədə olmaq və təlimatlandırmaq üçün göstərişlərdən necə istifadə etmək barədə təxmini fikir vermək üçün göstərişlərin əsaslarını əhatə edir.

Bütün nümunələr başqa cür göstərilmədiyi təqdirdə [OpenAI's playground](https://platform.openai.com/playground) istifadə edərək `text-davinci-003` ilə sınaqdan keçirilir. Model standart konfiqurasiyalardan istifadə edir, yəni `temperature=0,7` və `top-p=1`.
